Contiki fork for the uunode

The uunode has the following radio modules and sensors:
- CC2650
- CC1310
- CC2500
- RC1180HP-TM
- same sensors as TI SensorTag 2.0 (CC2650)

For more information about the uunode, contact christian.rohner@it.uu.se
